Meridian is a clean, professional personal brand block theme for thought leaders, coaches, and executives.

== Description ==

Meridian is a Full Site Editing (FSE) block theme built for personal brand websites — ideal for thought leaders, leadership coaches, executives, authors, and consultants.

Features:
* Two-column layout with sticky sidebar for profile, bio, and newsletter opt-in
* Six fully customizable block patterns: Hero Bio, Expertise Grid, Experience Timeline, Talks Grid, Publications List, and Newsletter CTA
* Professional color system with primary, secondary, accent, success, warning, and error ramps plus full neutral scale
* Fluid typography using the Inter font family
* Multiple page templates: Default, No Sidebar, and Full Width
* Archive, Search, Single Post, and 404 templates included
* Accessibility-ready with skip links, keyboard navigation support, and WCAG AA color contrast
* Translation-ready with full .pot file
* No page builder required — works entirely with the WordPress Site Editor

== Frequently Asked Questions ==

= Does this theme require any plugins? =

No. Meridian works entirely with WordPress core blocks and the Site Editor. No plugins are required.

= How do I set up the sidebar with my profile photo? =

Go to Appearance > Editor > Template Parts > Sidebar. Click the profile image block and upload your photo. Update your name and bio in the text blocks below.

= How do I change the colors? =

Go to Appearance > Editor > Styles (the half-circle icon) > Colors. You can change the entire color palette from there, or customize individual block colors.

= How do I add my newsletter service? =

The subscribe forms use plain HTML. Go to Appearance > Editor > Template Parts > Sidebar, click the subscribe form block, and replace the form action URL with your newsletter provider's endpoint (Mailchimp, ConvertKit, etc.).

= Can I use this theme without the sidebar? =

Yes. When editing any page or post, use the Template selector in the sidebar to switch to the "No Sidebar" or "Full Width" template.

= Is this theme accessible? =

Yes. Meridian meets WordPress accessibility requirements: skip links are provided, all interactive elements support keyboard navigation with visible focus indicators, and the color palette maintains WCAG AA contrast ratios.
